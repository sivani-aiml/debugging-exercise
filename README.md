# Debugging Exercise

A simple Python project created to practice debugging skills and basic Python concepts.

## Features

* Calculate average score from a list
* Use functions
* Practice conditional statements
* Learn debugging techniques

## Technologies Used

* Python 3
* Git
* GitHub

## How to Run

1. Clone the repository:
   git clone https://github.com/sivani-aiml/Debugging-Exercise.git

2. Navigate to the project folder:
   cd Debugging-Exercise

3. Run the program:
   python main.py
